> 工廠點火 — CEO 就位，現在讓它開始賺錢
> 警告：90% 的人這一步順序做反，於是蓋了一座沒人買單的漂亮工廠 _

CEO 就位了。但一座沒在賺錢的工廠，不叫工廠，叫燒錢的玩具。

這一章只解決一件事：**從零開始，讓你的 AI 工廠賺到第一筆錢，再長成一條完整產線。**

而這一切的關鍵，是兩個字的順序：先 **SELL**，再 **BUILD**。

---

## CH05-1｜每座工廠都有兩個核心：BUILD + SELL

先記住一個鐵則。任何一條會賺錢的產線，都由兩個核心組成，缺一不可：

```
BUILD（建造）  ──  用 AI 做出產品（網站 / 工具 / 課程 / 服務）
SELL （銷售）  ──  用 AI 把產品賣出去、收到錢
```

| 缺了哪個 | 下場 |
|---|---|
| 只有 BUILD，沒有 SELL | 做出一堆東西，沒人買。倉庫滿了，戶頭空了。 |
| 只有 SELL，沒有 BUILD | 很會賣，但沒東西可賣。 |

很多人是「BUILD 巨人、SELL 侏儒」——花三個月做一個超精緻的產品，然後完全不會賣。

The Super One 的工廠，**兩個核心一樣壯**。而且——你會先練那個大家最害怕的：SELL。

---

## CH05-2｜為什麼從 SELL 起步

矽谷投資人 Naval 有一句被引用到爛、但真的對的話：

> **"Learn to Sell, Learn to Build."**
> 學會銷售，學會建造。

注意這個順序——**Sell 在前。**

不是因為 SELL 比 BUILD 重要，是因為 SELL 先做，能救你一命：

```
先 BUILD 再 SELL：
  花一個月做產品 → 上市 → 發現沒人要 → 一個月白燒

先 SELL 再 BUILD：
  先確認有人願意付錢 → 才動手做 → 做的是「已經有人要」的東西
```

先 SELL，你是在用市場的錢、市場的訊號來決定要蓋什麼。你不再賭，你在驗證。

> 起步原則：**先 SELL，先收到第一筆真錢，再放大 BUILD。**

**但這是軟建議，不是鐵則。** AI 已經把 BUILD 的成本壓到極低——最快一週就能做出一個能賣的產品。所以如果你手上已經有一個很有把握的東西，先 BUILD 出來再賣，也完全可以。重點不是死守順序，是**別在沒人要的東西上燒掉幾個月**。

---

## CH05-3｜SELL：先收到第一筆錢

「先賣」聽起來很抽象。具體就一句話：

> **在你把產品做完之前，先讓一個陌生人付錢給你。**

怎麼做？最快的路徑長這樣：

```
1. 用 AI 做一頁銷售頁（不是產品，是一頁說明「我要賣什麼、賣多少」）
2. 把它丟到你已有的受眾 / 社群 / 廣告面前
3. 開放預購、訂金、或早鳥
4. 看有沒有陌生人真的掏錢
```

關鍵是問對問題。**別問**「你覺得這個想法好嗎？」——所有人都會客氣地說不錯。

**要問**的是：「你現在願意付錢嗎？」

付錢，才算數據。說「很棒啊」不算，那是禮貌，不是需求。

收到第一筆陌生人的錢，你就拿到了整座工廠最珍貴的東西：**一個被市場驗證過、值得放大的方向。**

---

## CH05-4｜BUILD：把一人公司的職能，變成 Agent 工作流

有人付錢了。現在才動手蓋產線——而且不是你自己蓋，是把一人公司的每個職能，一條條交給 Agent。

一條完整產線要跑的事：

```
獲客 → 內容 → 銷售頁 → 成交 → 交付 → 客服 → 數據
```

過去這些全是你一個人扛。現在，每一格都變成一個 Agent 工作流，由你的 AI CEO 調度。

要讓這條產線跑起來，四大件就位（其中兩件你上一章已經備好）：

| 零件 | 角色 | 狀態 |
|---|---|---|
| **AI CEO**（Claude Code / Codex） | 運營整廠、調度 Agent | ✅ CH04 已任命 |
| **創辦人知識庫** | 讓全廠像「你」 | ✅ CH04 已起草 |
| **Agent 員工**（通用＋專精） | 執行各環節 | 本章開始配置 |
| **Agent 工作流** | 讓每件事自動跑 | 本章開始接線 |

你做的不是「自己跑這七格」，是「設計好這七格、交給 AI CEO，讓它帶 Agent 跑」。

跑通一次，你就有了**第一條會自己運轉的產線**。這條線，就是未來三位一體（品牌 × 媒體 × 電商）的第一條腿。

---

## CH05-5｜今天的第一步：先賣，別先做

如果你現在腦中有一個產品點子，先忍住「我要把它做完美」的衝動。

今天就做一件事：

> **用 AI 做一頁銷售頁，把你想賣的東西寫清楚，標上價格，丟到 10 個真實的人面前，問他們：「現在願意付錢嗎？」**

- 有人付 → 太好了，你拿到綠燈，下一步才動手 BUILD。
- 沒人付 → 更好，你只花了一頁的力氣，就省下幾個月的白工。改題目，再丟一次。

這就是 SELL → BUILD 的精神：**讓市場先點頭，你才出力。**
